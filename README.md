## 📋 Sobre o projeto

O **FinanceControl** é uma aplicação web front-end para controle de finanças pessoais. Permite visualizar o saldo, registrar receitas e despesas, e acompanhar as transações do mês em um dashboard intuitivo.

---

## 🖥️ Páginas

| Página | Descrição |
|---|---|
| `login.html` | Tela de autenticação com validação de e-mail e senha |
| `cadastro.html` | Formulário de criação de conta com máscara de CPF e telefone |
| `index.html` | Dashboard com cards de resumo financeiro e tabela de transações |

---

## ✨ Funcionalidades

- 📊 Dashboard com resumo de saldo, receitas, despesas e economia do mês
- ➕ Lançamento de novas transações diretamente no dashboard
- 🏷️ Categorias: Alimentação, Transporte, Saúde, Lazer, Salário e Outros
- ✅ Validação de formulários (login e cadastro) no front-end
- 📱 Layout responsivo para desktop e mobile

---

## 🎨 Design

Tema **Cyberpunk Dark** inspirado no GitHub Dark, com:

- Paleta baseada em `#0d1117` / `#161b22`
- Destaques em ciano `#00ffe7` e roxo `#b57bee` com efeito glow
- Tipografia com `JetBrains Mono` + `Inter`
- Scrollbar, badges e tabelas estilizados

---

## 🚀 Como usar

Não requer instalação. Basta abrir os arquivos no navegador ou usar a extensão **Live Server** no VS Code:

1. Clone o repositório:
   ```bash
   git clone https://github.com/MARISTELAOLIVEIRA/finance-control.git
   ```
2. Abra a pasta no VS Code
3. Clique com o botão direito em `login.html` → **Open with Live Server**
